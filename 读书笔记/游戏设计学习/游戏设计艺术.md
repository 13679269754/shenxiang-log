## 第二章 设计师创造体验

没有那两个人能对同一事物产生完全相同的体验 -- 没一个人对任何一件事的体验都是独一无二的。

**游戏不等于体验**
游戏设计师只在意那些看起来存在的东西。玩家和游戏是真实存在的，而体验是想象的。

**追寻彩虹的三种途径**
这里没有规则！我们只是在努力实现点什么 --托马斯·爱迪生 

心理学
* 行为学家
* 现象学家


人类学
文化人类学的方法：尝试吧自己融入人群。

设计学

**反思：力量，风险和实践**

风险1：导致对事实做出错误判断
当处理人们的内心思维，尝试理解体验，感受事物时，反思是一种强有力和可信赖的工具。作为游戏设计师，我们应当更加注重事物的感受而不是真实的正确。

风险2：自己的体验不一定适用于其他人
两个极端
“我只为与我一样的玩家设计游戏”
“不能相信任何个人的观点”
如果不采用极端的方式，你只有再次通过倾听才能摆脱这种危险。
想文化人类学家一样，把自己放在受众的位置。

**仔细分析你的想法**

剖析体验是不反思的主要目标--设计师必须要做的事。
事实上，你应该能够分析任何你可能产生的体验。你分析的越多，就越能清晰的知道你的游戏应该创造哪种体验。

```
1号透镜：情感

人们可能会忘记你所说的，但他们永远也不会忘记你带给他们的感觉。-- 玛雅·安吉罗

问自己几个问题：
* 我希望玩家体验到怎么样的情感？ 为什么？
* 当人们玩游戏时，他们产生了怎样的情感？为什么？
* 我怎样缩小实际体验与我的设想间的差距？
```

**战胜海森堡原理**

**本质体验**

关键就是：一个优秀的游戏不需要你完美复制真实的体验。你需要做的事为你的游戏找出这种体验的本质。

如果你不知道你想要什么，你就不会关注你得到了什么。但如果你确实知道你想要什么--你对自己的游戏能够给玩家们带来怎样的感受就会有一种愿景。

```
2号透镜：本质体验

* 我想要让玩家获得怎样的体验？
* 这种体验的本质是什么？
* 我的游戏如何抓住这种本质？
```

当你对了理想中的体验和它所需的元素有了清晰的认识时，你的设计就有了方向。

**你的感受都是真实的**

我们又知道我们的体验“并不是真的现实”。我们会用自己的意识和大脑过滤现实，实际体验到的意识是一种幻觉。但这种幻觉是我们所能拥有的最真实的东西。
